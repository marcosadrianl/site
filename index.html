Proyectos

Laboratorio físico: Cálculo del mu estático y dinámico de un sistemaLink

Mediante elementos mecánicos y electrónicos, desarrollamos un software en Arduino que permite registrar el ángulo de inclinación al momento de producirse el movimiento de la pieza. Luego, mediante el uso de Arduino y MyOpenLab se desarrolló un software que permite visualizar los datos registrados en tiempo real, así como también calcular el mu dinámico y estático de la pieza.

Desarrollo de software controlador para adquisidor de Datos P-rhoLink

Junto a un equipo de trabajo en el Laboratorio de Enseñanzas en Educación de las Ciencias, desarrollamos el software para el adquisidor de datos de P-rho, utilizando lenguaje Python. Este software permite a los usuarios visualizar los datos del P-rho en tiempo real, así como también medir valores de tensiones y corrientes, además de generar gráficas de las mismas.
