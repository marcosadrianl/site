Proyectos

Laboratorio físico: Cálculo del mu estático y dinámico de un sistemaLink

Mediante elementos mecánicos y electrónicos, desarrollamos un software en Arduino+firmata que permite registrar el ángulo de inclinación al momento de producirse el movimiento de la pieza. Luego, mediante el uso de Arduino y MyOpenLab se desarrolló un software que permite visualizar los datos registrados en tiempo real, así como también calcular el mu dinámico y estático de la pieza.

Desarrollo de software controlador para adquisidor de Datos P-rhoLink

Junto a un equipo de trabajo en el Laboratorio de Investigación en Educación de las Ciencias, desarrollamos el software para el adquisidor de datos de P-rho, utilizando lenguaje Python. Este software permite a los usuarios visualizar los datos del P-rho en tiempo real, así como también medir valores de tensiones y corrientes, además de generar gráficas de las mismas.

Diseño, desarrollo y creación de Circuitos PCB

De manera individual, me embarque en la investigación para desarrollar circuitos de audio de baja distorsión armónica, empleando software de diseño como EasyEDA y EAGLE para el diseño de circuitos. Luego, mediante distintos métodos, llevar adelante la creación y puesta en prueba de los mismos.

Entre los proyectos desarrollados, puedo mencionar filtros activos de segundo orden (Audio), Fuentes con bajo nivel de ruido (ripple) y Adaptadores de impedancia activos (Audio).
